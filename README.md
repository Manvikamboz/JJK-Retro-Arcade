# JJK Retro Arcade Portfolio

An interactive, retro-glowing Nintendo 3DS themed arcade portfolio dedicated to Satoru Gojo, Yuji Itadori, Nobara Kugisaki, and the world of **Jujutsu Kaisen**.

![Jujutsu Kaisen Arcade](https://media1.tenor.com/m/-KGzQiMihX8AAAAC/nobara-nobara-kugisaki.gif)

## Features & Interactive Zones

*   **3DS Console UI**: Fully styled console interface with bilingual translations (English / Japanese), dynamic sound effects, retro scanline filters, and upgrading stats (ATK, DEF, SPD) using coins earned from games.
*   **Retro Game (Zone 01)**: A side-scrolling retro runner where you guide Nobara Kugisaki, jumping over cursed energy obstacles, collecting coins, and leveling up.
*   **Space Shooter (Zone 02)**: Face off against flying cursed spirits, firing energy blasts, and unlocking hidden achievements.
*   **Memory Match (Zone 03)**: Test your memory by matching cursed symbols and characters to gain XP and coins.
*   **Story & Manga Log**: An interactive timeline detailing 8 major story arcs from the Jujutsu Kaisen manga/anime with dedicated animated GIFs, streaming page redirects (JioHotstar), and official manga buying links.

## Technology Stack

*   **Frontend**: React, Vite, TypeScript
*   **Styling**: Custom vanilla CSS retro theme, neon-glow keyframe animations
*   **Audio**: Integrated audio soundboards for coin collections, level-ups, and clicks

## Getting Started

To run this project locally, clone the repository and follow these commands:

```bash
# Install dependencies
npm install

# Start local development server
npm run dev

# Build the production application
npm run build
```
